VEGAPI

An accounting API for business apps developers

VEGAPI is an accounting API designed for programmers who want to create new business applications but do not wish to develop all the back-end functionality required to handle invoices, payments, purchase orders, deliveries, inventory control and accounting / tax treatment and reporting.

The VEGAPI API defines resources identified by URLs with a uniform design. These resources can be manipulated by HTTP(S) transactions carrying representations in JSON or XML format.

The initial version of the API will implement the following sets of resources:

  • /entities : Customers, suppliers, employees...
  • /products : Products and services bought/sold
  • /orders : Orders for products/services
  • /deliveries : Delivery of goods sent or received
  • /documents : Invoices, cash receipts, credit/debit notes...
  • /payments : Payments, receivables and funds transfers
  • /accounts : Cash and bank accounts
  • /books : Accounting transactions and reports
  • /tax : Tax submissions and reports
